# Arduino Traffic Light Simulation

## Description
This project simulates a traffic light system using an Arduino board and an LCD display. The system runs through various safety messages while displaying a countdown timer for each stage.

## Features
- Displays safety messages:
  - "FOLLOW THE RULES"
  - "WEAR HELMET"
  - "STOP LOOK GO"
  - "SEAT BELT SAVES"
- Countdown timer for each message
- Uses a 16x2 LCD display for showing messages and timers
  
## Components Required
- Arduino Uno 
- 16x2 LCD display
- 2 Traffic lights(available in protues)
